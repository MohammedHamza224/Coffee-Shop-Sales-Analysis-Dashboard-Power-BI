# ☕ Coffee Shop Sales Dashboard | Data Analytics Project with Power BI

A Power BI dashboard project designed to analyze coffee shop sales data and transform raw transactions into actionable business insights. The dashboard provides a comprehensive view of sales performance, customer purchasing patterns, product category trends, and store comparisons.
________________________________________
📊 Features

Total Sales & Revenue Analysis
Transaction and Quantity Tracking
Month-over-Month (MoM) Sales Growth
Product Category Performance Analysis
Store/Branch Comparison
Peak Sales Hours Identification
Weekday vs Weekend Performance Analysis
